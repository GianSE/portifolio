# Currículo — fontes LaTeX

O currículo em si agora é uma página do portfólio: **`/curriculo`** (código em
`src/pages/CurriculoPage.tsx`), com PT/EN e botão de download em PDF (via
impressão do navegador). O conteúdo vem do mesmo CMS do portfólio
(`content/about/index.md` + `content/experiences/`), então nunca diverge do
que está publicado no site.

Esta pasta guarda só as versões originais em **LaTeX** (`curriculo.tex` /
`curriculum.tex`), mantidas como referência caso seja necessário um PDF
formatado via LaTeX em vez do PDF gerado pelo navegador.
